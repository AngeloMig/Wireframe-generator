# Awesome DESIGN.md reference library

The complete VoltAgent collection is available at `.design-md/awesome-design-md/design-md/`.

Before a visual redesign or new page:

1. Read `.design-md/awesome-design-md/README.md` to see the available references.
2. Choose the closest product or brand reference (for example `shopify`, `figma`, `linear.app`, or `apple`).
3. Read that reference's `DESIGN.md` and use it as design direction only; preserve this project's existing functionality and brand identity unless the user explicitly asks for a rebrand.
4. Translate the reference into this codebase's tokens, components, and responsive patterns. Do not paste marketing copy or recreate protected logos/assets without permission.

This library complements the `impeccable` skill: use this skill for visual reference, then use `impeccable` for implementation quality and iteration.
